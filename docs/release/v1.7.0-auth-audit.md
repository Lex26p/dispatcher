# v1.7.0 — Auth/audit integration foundation

## Summary

This release introduces the auth/audit integration foundation for Dispatcher.

The new backend library `dispatcher-auth-audit` provides audit event modeling, audit result modeling, audit sink abstraction, in-memory audit sink, audit logger, authorization policy evaluation, authorization decision auditing, authenticated operation context, HTTP/API auth-audit adapter foundation, and runtime auth-audit adapter foundation.

This release does not yet enforce authorization in the live HTTP server or live `DispatcherRuntime`. It provides a tested foundation that can be wired into real runtime and API operations in later stages.
