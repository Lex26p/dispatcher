Accepted note:

The broad `TEMP` marker can produce false positives for normal words such as
`template`, `Attempt`, `attempted`, `temperature`, and `tag-temperature`.
The final audit uses word-boundary matching.

### Exception usage

Reviewed backend libraries for `throw`.

Result: passed

Accepted note:

The project uses `std::logic_error` in Result/context accessor methods to guard
invalid API usage, such as reading a value, endpoint, response, session, or
context from an object that does not contain one. This is accepted misuse
protection and is not considered normal control flow.
