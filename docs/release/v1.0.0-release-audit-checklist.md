## Accepted Limitations for v1.0.0

The v1.0.0 core dispatcher runtime release intentionally does not include:

- real HTTP server implementation
- real gRPC server implementation
- real database storage backend
- real external telemetry protocol adapter
- real notification delivery backend
- production authentication provider integration
- distributed clustering
- persistent audit log backend
- UI/frontend

These are outside the scope of the core dispatcher runtime foundation.
