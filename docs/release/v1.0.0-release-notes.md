# v1.0.0 Core Dispatcher Runtime

Tag: `v1.0.0-core-dispatcher-runtime`

## Summary

This release establishes the first stable core foundation of the dispatcher
runtime.

It includes the domain model, telemetry ingest pipeline, history pipeline,
alarm pipeline, runtime composition, storage interfaces, API façade foundation,
configuration import/export, alarm acknowledgement, alarm shelving/suppression,
telemetry adapter contracts, operator/auth model, notification routing,
transport adapter foundation, and production hardening models.

This release is focused on the core runtime and library architecture. It does
not include real network servers, real database storage, external telemetry
protocol implementations, or UI/frontend components.
